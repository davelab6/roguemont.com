# [Roguemont.com](http://roguemont.com)

The website for the Roguemont typeface.

## Getting started

```sh
# Install Hap
npm install -g harp

# Install dependencies
npm install

# Start Harp
npm start

# Visit http://localhost:9000
```

## Deploying

```sh
# Compile and deploy to roguemont.com
npm run deploy
```
